¿Qué es AuDproyect?


AuDproyect es una plataforma colaborativa que permite crear audioguías en el servidor de la aplicación para que los usuarios puedan disfrutar de un contenido global de audioguías desde sus dispositivos móviles android.
Este proyecto fue mi trabajo fin de grado.

¿Cómo funciona?


AuDproyect consta de una página web y una aplicación móvil. La página web permite a las empresas, entidades o usuarios colaborar con el proyecto, creando sus propias audioguías en sel servidor. La aplicación móvil ofrece distintas funcionalidades al usuario. Para comprender su funcionamiento y el propósito del proyecto se recomienda ver la presentación.

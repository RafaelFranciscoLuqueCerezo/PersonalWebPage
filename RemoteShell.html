¿Qué tiene de distinto este RemoteShell hecho con Python?


Este RemoteShell hecho en Python (Python 3.8.5) permite registrar las pulsaciones de teclado del ordenador remoto, a la vez que se puede descargar cualquier archivo del mismo en el directorio que se desee.

¿Cómo funciona?


El usuario tendrá que esperar que el código del script del cliente sea ejecutado en el ordenador objetivo. Una vez ejecutado, el usuario podrá ejecutar comandos remotamente desde su ordenador, al igual que poder visualizar un nuevo terminal donde aparecerán registradas todas las pulsaciones del ordenador objetivo.
